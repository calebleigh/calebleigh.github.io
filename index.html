Hello. My name is

Caleb Lee

My coding journey began 8 years ago. During college I studied Computer Science and wrote code in C#. I took a job as an architectural drafter after college. However, I quickly learned that web developement was where I wanted to be and learned HTML, CSS and Javascript on my own. My love for design and attention to detail make a great combination for the web.

I am a father of 2. Married to the love of my life. While I love computers, coding and video games, I also enjoy being outside camping and fishing.

Here are some of the skills I have learned and web projects I have done.

Here is my coding journey. Things I have learned and used to make the best web apps and experiences all in one.

HTML/CSS

Like most, two of the first languages I studied and used. I quickly fell in love with CSS and the posibilities it had. Many hours were spent tweaking colors and pushing pixels to find the perfect combinations.

Javascript

I quickly learned that animations were something I wanted my sites to have. I learned basic javascript and delved a bit deeper into Jquery. Using JS I was able to make my projects feel more alive and responsive.

React

After coming back to code more, React was king. After learning this framework, I developed and published a web app that has over 10,000 registered users and consistent daily users.

UI/UX

Using classic CSS and frameworks like Bootstrap, I have been able to create fluid/responsive applications for all users.

Git/Github

When working with other developers, I have used Git and Github to seemlessly work on projects together. I have commited my own sites and projects on Github as well.

Backend

While I am still learning the backend of web developement, I have used databases to store user information as well as use Google to authenticate these users. My guide site used Node to store this information.

RuneScape Ironman Guide

In this project I used a pastebin of information and made a webapp for it. I used Google to login and store user information and stored in Firebase. There is a color selector for user customization. When you highlight a word, it has a popup that searches the Wiki for terms a player may not know. A search bar that scrolls to the entry you are looking for.

LDG Utah

This project was for the architectural studio I worked for. I used Wordpress to make a theme for them so they could edit it in the future. While pretty straight forward, it was a good intro in Wordpress and the possibilities there.

Barber J's

This is a project currently underway. We are doing a complete site makeover and building it on Shopify. We are creating a custom theme and landing page. I took this project to learn a little more about shopify intergration.

Design


While learning code, the design has been one of my favorite things. I have enjoyed learning new ways to implement new technologies and styles to keep user interaction fresh and fun.

My architectural job I have done 3D renderings and custom images for clients. I also have been a freelance graphic designer for many years doing anything from logos to birthday cards.

Hand Drawing

While I still enjoy drawing in the traditional sense, I really enjoy starting my design with hand drawn elements. If custom work needs to be done, I like to have as much control over it as possible at the start to ensure a clean product.

Photoshop/Illustrator

3D/Blender

Freelancing
